# Profil Diri - Ivan Aditia Firmansyah

Halaman profil pribadi sederhana, dibuat sebagai latihan HTML, CSS, dan JavaScript dasar.

## Fitur
- Struktur HTML semantik (header, nav, main, footer)
- Layout responsif menggunakan Flexbox dan CSS Grid
- Galeri foto yang menyesuaikan ke 1 kolom di layar mobile
- Menampilkan daftar user dari API (fetch async/await)

## Teknologi yang Digunakan
- HTML5
- CSS3 (Flexbox, Grid, Media Query)
- JavaScript (DOM Manipulation, Fetch API)

## Cara Menjalankan
1. Clone atau download repository ini
2. Buka file `profil.html` langsung di browser (double-click atau drag ke browser)
